  # Task List App

A simple task management app built with React, Redux, and Express.js.

## Features

- Create, edit, and complete tasks
- Assign tasks
- Track task status
- View task details

### Prerequisites

- Node.js
- npm or yarn

## Built With

- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/)
- [Express.js](https://expressjs.com/)
